EC2 S3 VPC Linux Tasks

Amazon EC2

Definition: Virtual server in AWS.

Use Case: Host websites and backend applications.

Hands-On Steps

  1. Launch Instance
  2. Select Amazon Linux
  3. Choose t2.micro
  4. Allow HTTP & SSH
  5. Launch

Install Web Server

sudo yum install httpd -y
sudo systemctl start httpd
echo "AWS EC2 Working" | sudo tee /var/www/html/index.html

Interview Question

Q: What is Security Group?
A: Stateful firewall attached to instance.

Amazon S3

Definition: Object storage service.

Hands-On

  1. Create Bucket
  2. Upload index.html
  3. Enable Static Website Hosting

Interview Question

Q: Difference between S3 and EBS?
A: S3 = Object storage, EBS = Block storage.

Amazon VPC

Definition: Private network inside AWS.

Hands-On

  1. Create VPC (10.0.0.0/16)
  2. Create Public Subnet
  3. Attach Internet Gateway
  4. Add Route 0.0.0.0/0

Interview Question

Q: Security Group vs NACL?
A: SG = Stateful, NACL = Stateless.
